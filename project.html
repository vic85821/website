HackMD Folder API

Node.js / Sequelize eager loading

HackMD is an online collaborative editor. We add folder feature that makes users manage files more easier. The project includes API implementation and UI design. We use eager loading to decrease the query times and improve efficiency. (still wait to be merged ... )

Design and Implement High-Performance Concurrent Database

C / Lock-free Algorithm / Atomic Instruction

The project is forked from Jserv's github. In this experiment, We use B+tree to implement a database. There are 10 times improvements than original database in performance. B+ tree is a data structure which is usually used to store enormous data. The main issues are how to use lock and bulk-loading. Could take a look in https://github.com/vic85821/bplus-tree
